    hierarchy. Deep hierarchies are harder to understand and maintain.
  Number of Children (NOC): Number of direct subclasses. High NOC means the
    base class is heavily relied upon — changes break many subclasses.
  Lack of Cohesion of Methods (LCOM): Measures how well the methods of a class
    relate to each other. High LCOM means the class is doing too many things.
  Response for a Class (RFC): Number of methods that can be executed in
    response to a message. High RFC means high complexity and testing burden.
  Weighted Methods per Class (WMC): Sum of complexities of all methods.
    High WMC means the class is too complex and should be split.
